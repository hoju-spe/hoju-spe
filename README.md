## Projects

### 1. Maily - 업무 이메일 자동화 AI Agent 서비스

반복적인 업무 이메일 처리 과정을 자동화하기 위해 Gmail 기반 메일 수집, AI 도메인/의도 분류, 요약, 일정 추출, 답변 초안 생성을 제공한 AI Agent 서비스입니다.

- 기술: Java, Spring Boot, JPA, MariaDB, REST API, JWT/OAuth2, Gmail API, RabbitMQ, AWS, Docker
- 역할: 백엔드 API 개발, Gmail API 연동, 인증/인가 구현, RabbitMQ 기반 비동기 처리, AI 작업 상태 관리, 관리자 운영 모니터링 기능 구현
- 링크: [Repository](https://github.com/hoju-spe/email-agent_Backend_Server)

### 2. Study Linker - 로컬 스터디 매칭 플랫폼

커뮤니티와 오픈채팅방에 흩어진 스터디 모집 문제를 해결하기 위해 위치·관심사 태그·인기 기반으로 근거리 맞춤 스터디 그룹을 추천하는 로컬 학습 매칭 플랫폼입니다.

- 기술: Java, Spring Boot, JPA, MariaDB, React, JavaScript, REST API
- 역할: 스터디 추천 API 개발, 사용자/스터디 그룹 데이터 모델링, 위치·관심사 기반 추천 로직 구현, React 화면 연동
- 링크: [Repository](https://github.com/hoju-spe/study-linker-backend)

### 3. Vision Agent Backend MVP

이미지 분석 결과를 저장·조회하는 비전 에이전트 백엔드 MVP입니다. 이미지 업로드 후 YOLO 모델로 객체를 탐지하고, bbox, class_name, confidence 등 추론 결과를 API로 관리합니다.

- 기술: Python, FastAPI, PyTorch/YOLO, SQLite, SQLAlchemy, Docker, AWS EC2
- 역할: 이미지 업로드/추론 API 개발, 분석 결과 데이터 모델링, YOLO 추론 파이프라인 연동, Docker Compose 기반 EC2 배포
- 링크: [Repository](https://github.com/hoju-spe/vision-agent-backend-mvp)
